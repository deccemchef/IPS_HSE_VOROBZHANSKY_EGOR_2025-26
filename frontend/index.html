1. Монополия

Монополия — это рыночная структура, при которой товар или услугу предлагает один продавец, не имеющий близких конкурентов.

В стандартной модели монополист выбирает объём, исходя из условия MR = MC.

По сравнению с конкурентным рынком монополист продаёт меньше и по более высокой цене, что ведёт к появлению DWL.

2. Основные обозначения

MB(k) — предельная полезность k-й единицы товара.

V(q) — суммарная ценность q единиц товара.

V(q) = MB(1) + MB(2) + ... + MB(q)

T(q) — общий платёж покупателя за q единиц.

U(q) — чистая полезность покупателя.

U(q) = V(q) - T(q)

3. Выбор покупателя

Покупатель сравнивает варианты 0, 1, 2, ... единиц и выбирает тот объём, который максимизирует полезность.

q* = argmax U(q)

4. Ценовая дискриминация

Ценовая дискриминация — продажа одного товара разным покупателям по разным ценам.

Для её осуществления нужны: рыночная власть, возможность различать покупателей, ограничение перепродажи.

4.1. Ценовая дискриминация первой степени

Фирма знает готовность платить каждого покупателя и назначает индивидуальную оплату.

T(q) = V(q)

График PD1

4.2. Ценовая дискриминация второй степени

Двухчастный тариф: фиксированный взнос плюс цена за единицу.

T(0) = 0,   T(q > 0) = F + p · q

График PD2

4.3. Ценовая дискриминация третьей степени

Фирма делит покупателей на наблюдаемые группы и назначает каждой группе свою цену.

T(q) = p[segment] · q

График PD3

5. Экономические показатели

Revenue — выручка фирмы.

VarCost — переменные издержки: mc · Q.

Profit — прибыль:

Profit = Revenue - VarCost

CS — consumer surplus.

PS — producer surplus.

W — общественное благосостояние:

W = CS + PS

6. Связь теории с симулятором

Симулятор переводит микроэкономические идеи в вычислимую форму. Для каждого покупателя задаётся функция спроса, затем строится функция платежа T(q), после чего покупатель выбирает объём, который максимизирует его полезность.

Индивидуальные решения агрегируются в показатели фирмы и рынка: Q, Revenue, Profit, CS, PS, W.
